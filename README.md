# Paperie
Paperie는 찾고싶은 참고문헌 서비스를 한 번에 자동으로 생성해주는 서비스입니다.


## 핵심 기능 설명
- API를 사용하여 논문, 책, 뉴스의 출처 표기법에 따른 참고문헌 형식 제공
- 참고문헌 복사하기 기능 제공
- 마이페이지를 통한 인용 기록 열람 가능
- 웹사이트의 참고 문헌을 바로 생성해주는 크롬 확장 프로그램 제공
